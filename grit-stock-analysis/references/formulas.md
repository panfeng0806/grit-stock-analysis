# 衍生财务指标自动计算公式

## 强制规则

阶段2完成后、阶段3开始前，**必须逐项检查以下指标是否可从已有数据推算**。禁止在可以推算的情况下标注"待查"。

推算优先级：①已有数据直接算 → ②券商研报提取 → ③标注"推算值,置信度中"写入 → ④以上都失败才标"未识别"。

---

## 季度财报（当季缺失时）

```
Q4收入 = 审计后FY全年收入 − (Q1 + Q2 + Q3)
Q4毛利 = 审计后FY全年毛利 − (Q1毛利 + Q2毛利 + Q3毛利)
Q4净利 = 审计后FY全年净利 − (Q1净利 + Q2净利 + Q3净利)
Q4毛利率 = Q4毛利 ÷ Q4收入 × 100%
```

前提：有审计后FY全年数据 + 前三个季度Excel数据。

---

## 盈利能力

```
毛利率 = 毛利 ÷ 收入 × 100%
净利率 = 净利润 ÷ 收入 × 100%
EBIT Margin ≈ (净利润 + 利息支出 − 利息收入 − 所得税) ÷ 收入 × 100%
            或 营业利润 ÷ 收入

ROE = 净利润 ÷ 平均股东权益
    平均权益 = (期初权益 + 期末权益) ÷ 2

ROIC ≈ EBIT × (1 − 税率) ÷ 平均投入资本
    投入资本 = 股东权益 + 有息负债 − 现金 − 数字资产（加密货币公司）
    ⚠️ 如无税率数据，保守用 EBIT ÷ 投入资本
```

---

## 估值指标

```
市值 = 实时股价 × 总股本

PE-TTM = 实时股价 × 总股本 ÷ 最近4个报告季归母净利之和
    ⚠️ 全文只能出现一个PE-TTM值
    ⚠️ TTM净利含大额非经常项目（如BTC公允价值变动）时，PE失真，标注警告

PB = 实时股价 ÷ BVPS
    BVPS = 最新股东权益 ÷ 总股本

P/S = 市值 ÷ 最近一个FY收入

EV = 市值 − 净现金
    净现金 = 现金 + 数字资产 − 总负债
    ⚠️ 加密货币公司的数字资产归入净现金计算

EV/Adj.EBITDA = EV ÷ 最新FY Adj.EBITDA
```

---

## 费用率

```
销售费用率 = 销售费用 ÷ 收入 × 100%
管理费用率 = 管理费用 ÷ 收入 × 100%
研发费用率 = 研发费用 ÷ 收入 × 100%
期间费用率 ≈ 毛利率 − EBIT Margin
```

---

## 运营指标

```
自挖矿平均成本/BTC = (自挖矿COGS + 相关运营费用) ÷ 自挖BTC产量
云挖矿盈亏平衡/BTC = (云挖矿COGS + 相关运营费用) ÷ 云挖BTC产量
```

⚠️ 如20-F/年报中已披露，直接引用。未披露则标注"未识别"。

---

## 每股指标

```
EPS = 归母净利润 ÷ 加权平均普通股股数
BVPS = 股东权益 ÷ 期末普通股股数
总股本反推 = 归母净利 ÷ EPS（当季报有EPS无股数时）
```

---

## 增长率

```
同比增长率 = (当期 − 去年同期) ÷ |去年同期| × 100%
环比增长率 = (当期 − 上期) ÷ |上期| × 100%
```

## 股权融资关键参照

```
PIPE/IPO投资者成本 vs 现价 → 计算稀释幅度
ATM增发均价 vs PIPE价格 → 判断融资成本趋势
储架注册剩余空间 ÷ 现价 → 估算潜在稀释股数
认股权证行权价 vs 现价 → 判断现金流入可能性
```
