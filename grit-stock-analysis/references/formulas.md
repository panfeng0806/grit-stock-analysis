# 财务指标自动计算 + Wind Excel 数据提取

## 第一部分：衍生指标计算公式

阶段2完成后、阶段3开始前，**必须逐项检查以下指标是否可从已有数据推算**。禁止在可以推算的情况下标注"待查"。

推算优先级：①已有数据直接算 → ②券商研报提取 → ③标注"推算值,置信度中"写入 → ④以上都失败才标"未识别"。

> ⚠️ **扣非口径**：当公司存在重大非经常性损益时，所有指标必须同时提供GAAP和扣非两套数字。Wind的「扣非后归母净利」行不可盲信——详见 `references/non-recurring-adjustment.md`。

---

### 季度财报（当季缺失时）

```
Q4收入 = 审计后FY全年收入 − (Q1 + Q2 + Q3)
Q4毛利 = 审计后FY全年毛利 − (Q1毛利 + Q2毛利 + Q3毛利)
Q4净利 = 审计后FY全年净利 − (Q1净利 + Q2净利 + Q3净利)
Q4毛利率 = Q4毛利 ÷ Q4收入 × 100%
```

前提：有审计后FY全年数据 + 前三个季度Excel数据。

---

### 盈利能力

```
毛利率 = 毛利 ÷ 收入 × 100%
净利率 = 净利润 ÷ 收入 × 100%
EBIT Margin ≈ (净利润 + 利息支出 − 利息收入 − 所得税) ÷ 收入 × 100%
            或 营业利润 ÷ 收入

ROE = 净利润 ÷ 平均股东权益
    平均权益 = (期初权益 + 期末权益) ÷ 2

ROIC ≈ EBIT × (1 − 税率) ÷ 平均投入资本
    投入资本 = 股东权益 + 有息负债 − 现金 − 数字资产（加密货币公司）
    ⚠️ 如无税率数据，保守用 EBIT ÷ 投入资本
```

---

### 估值指标

```
市值 = 实时股价 × 总股本

PE-TTM = 实时股价 × 总股本 ÷ 最近4个报告季归母净利之和
    ⚠️ 全文只能出现一个PE-TTM值
    ⚠️ TTM净利含大额非经常项目（如BTC公允价值变动）时，PE失真，标注警告

PB = 实时股价 ÷ BVPS
    BVPS = 最新股东权益 ÷ 总股本

P/S = 市值 ÷ 最近一个FY收入

EV = 市值 − 净现金
    净现金 = 现金 + 数字资产 − 总负债
    ⚠️ 加密货币公司的数字资产归入净现金计算

EV/Adj.EBITDA = EV ÷ 最新FY Adj.EBITDA
```

---

### 费用率

```
销售费用率 = 销售费用 ÷ 收入 × 100%
管理费用率 = 管理费用 ÷ 收入 × 100%
研发费用率 = 研发费用 ÷ 收入 × 100%
期间费用率 ≈ 毛利率 − EBIT Margin
```

---

### 运营指标

```
自挖矿平均成本/BTC = (自挖矿COGS + 相关运营费用) ÷ 自挖BTC产量
云挖矿盈亏平衡/BTC = (云挖矿COGS + 相关运营费用) ÷ 云挖BTC产量
```

⚠️ 如20-F/年报中已披露，直接引用。未披露则标注"未识别"。

---

### 每股指标

```
EPS = 归母净利润 ÷ 加权平均普通股股数
BVPS = 股东权益 ÷ 期末普通股股数
总股本反推 = 归母净利 ÷ EPS（当季报有EPS无股数时）
```

---

### 增长率

```
同比增长率 = (当期 − 去年同期) ÷ |去年同期| × 100%
环比增长率 = (当期 − 上期) ÷ |上期| × 100%
```

---

### 股权融资关键参照

```
PIPE/IPO投资者成本 vs 现价 → 计算稀释幅度
ATM增发均价 vs PIPE价格 → 判断融资成本趋势
储架注册剩余空间 ÷ 现价 → 估算潜在稀释股数
认股权证行权价 vs 现价 → 判断现金流入可能性
```

## 第二部分：Wind/Choice 财务摘要 Excel 数据提取

> ⚠️ **核心陷阱**：Wind 导出的财务摘要 Excel 通常 80-90 行，前 30 行仅是利润表+资产负债表。「关键比率」区块（ROE/ROIC/毛利率/净利率/PE/PB/PS）在 31-87 行。永远 `max_row` 全量读取！

### Excel 行区块分布（标准 Wind 导出模板）

| 行号 | 区块 | 关键字段 |
|:---|:---|:---|
| 1-4 | 标题头+报告期 | 股票名称、代码、单位、日期列 |
| 5-20 | **利润表摘要** | 营业总收入(含同比)、营业利润、净利润、**归母净利润**(含同比)、**扣非归母**(含同比)、研发支出、EBIT、EBITDA |
| 26-40 | **资产负债表摘要** | 流动资产/固定资产、**资产总计**(含同比)、负债合计(含同比)、股东权益/归母权益(含同比) |
| 45-55 | **现金流量表摘要** | **经营活动现金净流量**(含单季度!)、购建固定资产、筹资活动净流量、折旧与摊销 |
| 59-70 | ⭐ **关键比率** ⭐ | **ROE(摊薄)**、ROE(加权)、扣非ROE、**ROA**、**ROIC**、**销售毛利率**(含单季度!)、**销售净利率**(含单季度!)、EBIT Margin、**资产负债率**、资产周转率 |
| 75-83 | **每股指标** | EPS(基本/稀释/摊薄)、扣非EPS、BPS、SPS、OCFPS、每股自由现金流 |
| 86-87 | **估值指标** | **P/E(TTM)**、P/E(LYR)、**P/B(MRQ)**、**P/S(TTM)** |

### 关键陷阱

1. **「关键比率」区块（行59-70）是GRIT最核心数据源**——ROIC、季度毛利率、季度净利率全在这里，前30行完全看不到
2. **现金流量表含单季度经营现金流**——可分析季度波动
3. **估值指标基于Wind导出日股价**，非实时——仅用于竞争对比的同一时点相对估值，报告优先用实时API
4. **竞争对手Excel结构完全相同**——底部同样有P/E/PB/PS

### 读取代码模板

```python
import openpyxl

wb = openpyxl.load_workbook(path, data_only=True)
ws = wb['sheet1']  # Wind导出通常用sheet1

# 永远用 max_row 全量读取
for row in ws.iter_rows(min_row=1, max_row=ws.max_row, values_only=False):
    vals = [str(cell.value) if cell.value is not None else '' for cell in row]
    if vals[0].strip():
        print(' | '.join(vals))
```

### 跨公司列映射（不统一时的处理）

不同公司 Wind Excel 年报列位置可能不同。禁止硬编码列号，必须程序扫描：
```python
# 找到 row[2] == '12个月' 的所有列 → 取前4个为最新4年年报列
year_cols = [i for i, v in enumerate(row) if str(v).strip() == '12个月']
latest_4 = year_cols[:4]  # 按日期降序，前4个为最新4年
```

### 港股特殊：币种标注

### Wind 主营构成分类局限

Wind 可能将国际业务收入全部标为"中国大陆100%"，这是 Wind 分类方式导致。必须从年报/研报提取国际业务数据。
