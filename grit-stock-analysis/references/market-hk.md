# 港股市场参考（v2.3实战提炼）

## 港股年报结构

港股年报（港交所披露易下载）的章节结构与SEC 20-F高度相似，包含等效的Item 4-7和F-pages：

| 港股年报章节 | 等效SEC | 内容 | 提取方式 |
|:---|:---|:---|:---|
| 董事及高級管理人員 | Item 6.A | 姓名/年龄/职务/履历 | grep '董事及高級管理人員.*年齡.*職位' |
| 股份所有權 (E) | Item 7.A | 主要股东/持股明细/百分比 | grep 'E.*股份所有權\|主要股東' |
| 股份激勵計劃 | Item 6.B | 期权池/授予表/SBC细节 | grep '全球激勵\|股份獎勵\|購股權計劃' |
| 財務報表附註 | F-pages | 审计后BS/IS/CF + 详细附注 | pdftotext后定位 |
| 管理層討論與分析 | MD&A | 业务回顾/风险/前景 | 全文浏览 |

## 双上市处理（如 0000.HK + TICKR.US）

**核心原则：港股年报章节=20-F结构，分析港股时不需要额外拿SEC 20-F，两者内容重复。**

| 场景 | 处理方式 |
|:---|:---|
| 用户明确说"分析港股0000" | 仅用港股年报，不取20-F |
| 用户明确说"分析美股TICKR" | 用20-F（按market-us.md流程） |
| 用户要求双市场对比 | 港股年报为主，20-F仅用于交叉验证 |

## 实时行情

### Yahoo Finance 港股代码格式

**关键：港股代码去前导零！**

```
✅ 0000.HK  — 正确
❌ 00000.HK — 错误（返回"symbol may be delisted"）
✅ 0005.HK  — 汇丰控股
```

### 获取行情

```bash
# chart端点（可用，不需cookie）
curl -sL "https://query1.finance.yahoo.com/v8/finance/chart/0000.HK?interval=1d&range=1mo" \
  -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64)"

# quoteSummary端点（被Yahoo封禁，需cookie）
# ❌ 不要用: query2.finance.yahoo.com/v10/finance/quoteSummary
```

**PE/市值必须自算**（Yahoo不返回），公式见 `references/formulas.md`。

## Wind Excel 港股特殊性

### 币种标注行（行65-68）

港股公司Wind导出Excel底部有币种元数据：

| 行 | 内容 | 含义 |
|:---|:---|:---|
| 65 | 显示币种: HKD | 表中数字以HKD显示 |
| 66 | 原始币种: CNY | Wind内部从CNY转换 |
| 67 | 转换汇率: 0.87 | CNY × 1/0.87 = HKD（≈1.149） |
| 68 | 汇率类型: 最新汇率 | 导出时的快照汇率 |

⚠️ Wind的0.87转换率（CNY→HKD）与市场汇率~1.08有显著偏差。用户通常要求以显示币种为准不手动矫正。

### 主营构成「地区100%中国大陆」陷阱

对于国际业务公司，Wind主营构成的地区分类往往全部标为"中国大陆100%"。**这不是数据错误，而是Wind分类方式导致**——必须从年报/研报提取国际收入占比补充。

### 竞争对手Excel

竞争对手的Wind Excel列映射需**程序化扫描**（见主skill pitfall 35）。

## 券商研报币种混合

中文券商（中泰/东吴/国信等）EPS预测以**人民币（元/股）**呈现。报告最终统一为HKD时需标注原始币种，转换可用Wind汇率或用户指定汇率。
