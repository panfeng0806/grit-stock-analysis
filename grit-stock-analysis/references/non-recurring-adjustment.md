# 非经常性损益剔除与扣非指标重算

> 版本：v1.0 | 创建：2026-05-12 | 来源：港股实战案例

## 触发条件

当以下任一信号出现时，必须启动扣非流程：

| 信号 | 典型案例 | 判断阈值 |
|:---|:---|:---|
| 净利润YoY ≫ 营收YoY | 净利+95% vs 营收+17% | 差距>30pct |
| 单季度净利异常 | Q3净利227亿 vs 正常~50亿 | 高出均值3x+ |
| 税前利润 - 营业利润巨大 | 444 - 180 = 264亿 | 差距>营收30% |
| OCF/NP断崖下跌 | 2.22x→0.43x | 下降>50% |

## 步骤A：定位非经常项目金额（年报）

```
# 搜索年报中的出售/处置描述
pdftotext -layout 年报.pdf - | grep -i "處置\\|處分\\|出售\\|Disposal\\|Gain on sale\\|投资收\\|投资收益"

# 关键：找到「其他（支出）/收入」附注中的最大单笔收益
# 案例：年报 line 3848-3851 明确写「确认了人民币152亿元的投资收益」
```

⚠️ **Wind「扣非后净利润」不可信**。实战案例中Wind扣非=GAAP完全一致，因为Wind未识别该非经常项目。**必须自主从年报提取原始金额**。

## 步骤B：定位相关税费

```
# 现金流量表附注中搜索资本利得税
pdftotext -layout 年报.pdf - | grep -i "處置權益法投資的資本利得稅\|capital gains tax"

# 现金流量表 line 9461 「處置權益法投資的資本利得稅 2,073」（RMB百万）
# → RMB 20.73亿 ≈ 23.83亿HKD（汇率0.87）
```

同时检查年报「所得税」附注中的有效税率描述，验证一致性。

## 步骤C：币种转换（港股/美股关键）

Wind Excel底部必读三行：
- 显示币种（如HKD）
- 原始币种（如CNY/RMB）
- 转换汇率（如0.87）

年报披露通常用原始币种（RMB），Excel显示用显示币种（HKD）。
转换公式：`HKD金额 = RMB金额 / 汇率`

## 步骤D：重算扣非指标（完整清单）

假设：非经常收益 = G（税前），关联税费 = T

### 利润表调整
```
扣非税前利润 = GAAP税前利润 - G
扣非所得税   = GAAP所得税 - T
扣非净利润   = GAAP净利润 - (G - T)
```
⚠️ 营业利润不受影响（非经常项目计入「其他收入」而非营业利润）

### 比率重算
```
扣非净利率 = 扣非净利润 / 营业收入
扣非净利YoY = (扣非净利 - 上年净利) / |上年净利|
扣非ROE = 扣非净利润 / 平均归母权益
```

### ROIC重算（比例法）
```
# 已知 GAAP ROIC = GAAP_EBIT × (1-t) / IC
# GAAP_EBIT 含非经常收益 G（Wind的EBIT定义可能包含投资收益）
扣非EBIT = GAAP_EBIT - G
扣非ROIC = GAAP_ROIC × (扣非EBIT / GAAP_EBIT)
```

### 估值重算
```
扣非PE-TTM = 市值 / 扣非TTM净利润
扣非EV/EBITDA = EV / (GAAP_EBITDA - G)
扣非OCF/NP = 经营现金流 / 扣非净利润
```

### 季度调整
定位非经常项目发生的具体季度，仅调整该季度：
```
扣非Q净利 = GAAP_Q净利 - (G - T)  # 仅影响发生季度
扣非Q净利率 = 扣非Q净利 / Q营收
扣非Q净利YoY = (扣非Q净利 - 上年同季净利) / |上年同季净利|
```

## 步骤E：级联更新报告（完整清单）

扣非不是只改财务表。以下**所有章节**都会受到影响：

| 章节 | 受影响内容 | 典型变化 |
|:---|:---|:---|
| 摘要 | 净利/PE/ROIC数字 + 加扣非说明框 | PE 7.65→12.67x |
| 核心财务指标表 | 加「2025A扣非」列 | 12个指标改 |
| 季度财务追踪 | Q列标「扣非」+ 净利/净利率/YoY/环比 | Q3 227→76.53亿 |
| 增长类型判定 | 盈利匹配类型从「利润>收入」→「利润≈收入」 | 全部改判定 |
| 竞争要素对比表 | ROIC/ROE/净利率/PE | 目标公司栏全换 |
| 资本要素指标表 | ROIC/ROE/OCF + 数据说明重写 | 3行+说明 |
| 资本要素综合评估 | 效率评级+现金流评级+发展制约判断 | 全部下调 |
| 估值-盈利预测 | 2025A标注扣非 + 加GAAP参考行 | 结构重排 |
| 估值-PE对比 | 可比公司表用扣非PE | 7.65→12.67x |
| 估值-目标PE | 估值差从4.5x→2.7x | 锚定调整 |
| 估值-3年1变2 | 净利增长从+40%→+22%，判定翻转为不符合 | **判定反转** |
| 估值-买入建议 | 预期收益率下调 | 17-24%→14-18% |
| 风险-退出风险 | 业绩描述更新 | 措辞修改 |
| 综合结论 | 全部核心数字+投资逻辑 | 5条→6条 |
| 前文结论+后文索引 | R8/R9描述更新 | 多处引用 |

## 步骤F：报告呈现规范

### 摘要扣非说明框
```
> 🔴 **扣非说明**：2025年XX出售[标的]（对价X美元），确认投资收益RMB X亿（≈HKD X亿），
> 相关资本利得税RMB X亿（≈HKD X亿）。本报告提供GAAP与扣非双口径。
```

### 财务表双列格式
```
| 指标 | 2023A | 2024A | 2025A (GAAP) | 2025A (**扣非**) |
```

### 估值标注
- 主用扣非PE：`扣非PE-TTM=12.67x`
- GAAP参考：`（GAAP 7.65x）`
- 明确标注：「扣非口径」「GAAP口径」

### 旧数字保留规则
⚠️ GAAP数字不能直接删除。保留作为对比参考，但每个出现位置必须有「GAAP」标注。

## 常见陷阱

1. **Wind扣非不可信**：必须自己算，年报是唯一权威来源
2. **币种混淆**：年报用RMB，Excel可能用HKD，必须确认汇率
3. **EBIT被污染**：Wind的EBIT可能包含投资收益（424.11含了174.71亿），需手动剔除
4. **季度定位错误**：非经常项目可能跨季度（很少见），确认发生季度
5. **税费不全**：只减收益不减税→高估扣非净利；必须找出现金流量表中的直接税费
6. **只改净利不改比率**：ROE/ROIC/PE全部需重算，否则估值判断完全失真
7. **COGS不受影响**：非经常收益不计入营业成本，毛利率不变
