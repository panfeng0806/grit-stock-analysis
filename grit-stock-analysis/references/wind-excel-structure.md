## 关键陷阱

1. **「关键比率」区块（行59-70）是GRIT分析最核心的数据源**——ROIC、季度毛利率、季度净利率、资产周转率、现净比全在这里，但这些数据在前30行完全看不到。
2. **现金流量表摘要（行45-55）含单季度经营现金流**——可用来分析季度现金流波动。
3. **估值指标（行86-87）基于Wind数据导出日股价**，非实时，仅用于竞争对比的「同一时点」相对估值。
4. **竞争对手Excel结构完全相同**——绿联/传音等Excel也用相同模板，底部同样有P/E/PB/PS。

## 读取代码模板

```python
import openpyxl

wb = openpyxl.load_workbook(path, data_only=True)
ws = wb['sheet1']  # Wind导出通常用sheet1

# 永远读取全部行
for row in ws.iter_rows(min_row=1, max_row=ws.max_row, values_only=False):
    vals = [str(cell.value) if cell.value is not None else '' for cell in row]
    if vals[0].strip():
        print(' | '.join(vals))
```
